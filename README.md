# Calculator CLI

A simple command-line calculator written in Python.

This project demonstrates basic Python programming concepts including:

- Functions
- User input
- Control flow
- CLI interaction

---

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Menu-based interaction

---

## Tech Stack

Python 3

---

## Installation

Clone the repository:

git clone https://github.com/william20000904-bit/calculator-cli-1.git

---

## Run the program

python calculator_cli.py

---

## Example

Enter option: 1  
Enter first number: 5  
Enter second number: 3  

Result: 8
